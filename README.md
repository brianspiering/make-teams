Make pairs and teams for quizzs and "pair" programming labs
=====

Usage:

`$ python make_teams.py`

Reads in `students.txt`

Saves a series of Markdown files that generate tables like this:

|     | Team 0 |     |     |
|:----:|:---:|:---:|:---:| 
| Pair 0 | Barbara | Dennis | |
| Pair 1 | Donald | Ada | |

|     | Team 1 |     |     |
|:----:|:---:|:---:|:---:| 
| Pair 2 | Marvin | Grace | |
| Pair 3 | Edsger | Alan | John |
